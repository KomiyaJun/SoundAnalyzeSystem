# 作品タイトル 概要
***タイトル:***SoundAnalyzeSystemDemo
***概要***パート別(ドラムやベースなど)再生を行える機能などを持ったSoundManagerを利用し、BGMのリアルタイムFFT解析を通じて、音と連動変形するオブジェクトやテンポ検知システムなどを構築したプロジェクトです。

# 主な機能と特徴
***プロジェクトの主な機能と特徴*** 
本プロジェクトでは、楽曲（BGM）を単に再生するだけでなく、オーディオデータをリアルタイムに解析し、ゲーム内の通知システムやオブジェクトの視覚的変化と高度に同期させるオーディオ・リアクティブ・システムを構築しています。

***1. レイヤー別・インタラクティブなサウンド管理 (Sound フォルダ)***
再利用可能なサウンドマネージャー: 

**UnitySoundManager.cs**
 を中心に実装し、複数音源の再生・停止・ボリューム管理を安全に行う仕組みを構築しています。
パート（レイヤー）別の動的制御: BGMを単一のファイルとして扱うのではなく、パート（ドラム、ベース、メロディ等）ごとに分割して同期再生しています。さらに、状況に応じて特定のパートのみを有効・無効化（TogglePart / AddPart / RemovePart）できるインタラクティブなサウンドシステムを実現しています。

**2. FFTを用いたリアルタイムオーディオ解析 (SoundAnalyzer フォルダ)**
高速フーリエ変換（FFT）の活用: 

**AudioAnalyzer.cs**
**AudioMixerAnalyzer.cs**
 により、再生中の音声からサンプリングした波形データをFFT解析し、周波数スペクトル（SpectrumData）をリアルタイムに抽出しています。
帯域（バンド）別の抽出処理: 単なる全体の音量ではなく、指定した周波数帯の平均音量を取得する機能（GetBandAverage(min, max)）を実装。これにより、「低音域（キック・ベース）」や「高音域（シンバル）」といった特定サウンドに絞った緻密な値が取得可能になっています。

***3. 音声解析データを活用したゲームギミック・視覚表現の同期***
解析された周波数成分やビートのデータを他クラス（UIやオブジェクト）に渡し、音楽と完全にリンクした表現を実現しています。

BPM・テンポ検知による通知システム: 

**BpmAnalyzer.cs**
を用いて、特定の周波数のピークを検知。楽曲のビートやテンポのタイミングに合わせて、ゲーム内の通知イベント等を正確に発火させます。
オーディオ・リアクティブなオブジェクト変化: 

**AmbientParticleSync.cs**
や 
**Visualizer.cs**
などを通じ、取得した低周波・高周波のデータに基づいてパーティクルの生成量、オブジェクトのトランスフォーム（拡大縮小）、発光（BeatLightEffect.cs）を連動させています。
